## Pasos a seguir para utilizar el repositorio

1. Clonar el proyecto
2. Instalar dependencias
```
$ yarn install
```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Cambiar las variables de entorno
5. Levantar la Base de Datos
```bash
$ docker-compose up -d
```
6. Levantar API en desarrollo
```bash
$ yarn run start:dev
```


## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
